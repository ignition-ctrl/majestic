# Majestic dotfiles (Arch Linux)
Dotfiles do Arch Linux

## Uso
 - `git clone https://github.com/sistematico/majestic ~/.dotfiles`
 - `pacman -S stow`
 - `cd ~/.dotfiles`
 - `stow bspwm`
 - `stow sxhkd`
 - `sudo stow -t /etc etc`
